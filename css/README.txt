1. Common `block` elements - can change width , not same line
  - paragraps <p>
  - headers <h1> to <h6>
  - divisions <div>
  - list and list items <ol> , <ul> , <li>
  - forms <form>
2. Commin `inline` elements - can be on same line , cant change width
  - spans <span>
  - images <img>
  - anchors <a>
3. `inline-block` - same line + change width
4. none

- visibility: hidden; -- makes the element hidden , but keeps its original position.

5. position:
  - static  - default
  - relative  - top: 50px; -- it does not affect position of anything else
  - absolute  - relative to its parent!!! ( does affect another elements)
  - fixed - will stay on scrolling.
